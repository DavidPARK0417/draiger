## 개요

### 목적

사용자들이 서비스 개발을 응원할 수 있도록 후원하기 기능을 추가합니다. 헤더 메뉴의 "문의하기"를 "후원하기"로 변경하고, 2가지 후원 플랫폼(Buy Me a Coffee, Toss 송금)을 한 페이지에 모두 제공합니다.

### 주요 특징

- ✅ 2가지 후원 플랫폼 통합 제공
- ✅ Toss 송금: QR코드 이미지 표시 및 모달 확대 보기
- ✅ 반응형 디자인 (모바일/태블릿/노트북/데스크탑)
- ✅ 다크모드 지원
- ✅ 아이콘 기반 직관적인 UI
- ✅ 외부 링크는 새 탭에서 열기 (사용자 경험 최적화)
- ✅ 디자인 시스템 준수

## 후원 플랫폼 비교

### 1. Toss 송금 (토스) QR코드 송금

**특징:**

- 가장 한국적이면서도 깔끔한 방법
- 토스 앱 사용자에게 최적화
- QR코드 이미지로 간편하게 후원 가능

**장점:**

- 수수료 없음
- 한국인이라면 누구나 토스 앱으로 즉시 송금 가능
- 별도의 후원 플랫폼 가입 절차 불필요
- QR코드 스캔만으로 간편하게 송금
- 모바일에서 토스 앱으로 바로 연결

**단점:**

- 토스 앱이 없는 사용자는 사용 불가
- 해외 사용자에게는 접근 불가
- QR코드 이미지 파일 관리 필요

**적용 방법:**

```
1. 토스 앱 실행
2. '송금' 메뉴 선택
3. '내 QR코드' 또는 'QR코드 보기' 선택
4. QR코드를 이미지로 저장
5. 저장한 이미지를 프로젝트에 추가 (public/images/toss-qr.png)
```

**아이콘:** Wallet 💳

---

### 2. Buy Me a Coffee (글로벌 감성 + 심플함)

**특징:**

- 글로벌 사용자에게 친숙한 플랫폼
- "커피 한 잔 사주기"라는 심리적 부담 없는 문구
- 세련된 디자인

**장점:**

- 디자인이 매우 세련됨
- 사용자에게 심리적 부담이 적음
- 현재 사이트의 UI/UX와 가장 잘 어울림

**단점:**

- 해외 서비스로 카드 번호 입력 필요 (이탈 가능성)
- 수수료 발생

**적용 방법:**

```
https://buymeacoffee.com/아이디
```

**아이콘:** Coffee ☕

                    fill
                    className="object-contain rounded-lg"
                    sizes="(max-width: 640px) 100vw, 400px"

                  토스 앱에서 QR코드를 스캔하여 송금해주세요
                </p>
              </div>
            </div>
          </div>
        )}

        {/* 감사 메시지 */}
        <div
          className="
          mt-12 sm:mt-16
          text-center
          p-6 sm:p-8
          bg-white dark:bg-gray-800
          border border-gray-200 dark:border-gray-700
          rounded-2xl
          shadow-md
        "
        >
          <Heart
            size={32}
            className="
              mx-auto mb-4
              text-emerald-500 dark:text-emerald-400
            "
          />
          <p
            className="
            text-base sm:text-lg
            text-gray-700 dark:text-gray-300
            leading-relaxed
          "
          >

            <br className="hidden sm:block" />더 나은 서비스를 위해 계속 노력하겠습니다.
          </p>
        </div>
      </div>
    </div>
  );
}
