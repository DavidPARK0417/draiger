## 2. 디자인 시스템 핵심 요소

### 2.1 색상 시스템

#### Primary Color: Emerald Green (#10B981)
- **용도**: 주요 버튼, 강조 요소, 성공 상태
- **호버**: #059669
- **액티브**: #047857
- **Tailwind**: `bg-emerald-500`, `text-emerald-500`

#### Secondary Color: Amber Yellow (#FBBF24)
- **용도**: 대기/경고 상태, 보조 강조
- **Tailwind**: `bg-amber-400`, `text-amber-400`

#### Danger Color: Red (#EF4444)
- **용도**: 오류, 삭제, 위험 액션
- **Tailwind**: `bg-red-500`, `text-red-500`

#### Neutral Colors: Gray Scale
- **텍스트**: #111827 (primary), #6B7280 (secondary), #9CA3AF (tertiary)
- **배경**: #F8F9FA (page), #FFFFFF (card)
- **테두리**: #E5E7EB (light), #D1D5DB (medium)

### 2.2 타이포그래피

#### 폰트
- **Primary**: Pretendard Variable (한글 최적화)
- **Monospace**: 시스템 모노스페이스 폰트
- **CDN**: `https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/variable/pretendardvariable.min.css`

#### 폰트 크기
- **xs**: 12px (작은 라벨)
- **sm**: 14px (테이블 데이터, 폼 라벨)
- **base**: 16px (본문 텍스트)
- **lg**: 18px (강조 텍스트)
- **xl**: 20px (섹션 제목)
- **2xl**: 24px (페이지 제목)
- **3xl**: 30px (메인 헤딩)

### 2.3 간격 시스템

- **기본 단위**: 1 unit = 0.25rem = 4px
- **표준 간격**: 4px (1), 8px (2), 16px (4), 24px (6), 32px (8)
- **컴포넌트별**:
  - 버튼 패딩: px-4 py-2.5 (기본)
  - 카드 패딩: p-6 (24px)
  - 폼 필드 간격: gap-4 (16px)

### 2.4 그림자 & 효과

- **기본 카드**: `shadow-md`
- **호버 카드**: `shadow-xl`
- **3D 효과**: `shadow-[0_20px_50px_rgba(16,185,129,0.15)]`
- **트랜지션**: `transition-all duration-300`
- **호버 효과**: `hover:-translate-y-1`

### 2.5 컴포넌트 스타일

#### 버튼
- **Primary**: `bg-emerald-500 hover:bg-emerald-600`
- **Secondary**: `bg-gray-100 hover:bg-gray-200`
- **Ghost**: `bg-transparent hover:bg-emerald-50`
- **Danger**: `bg-red-500 hover:bg-red-600`
- **모서리**: `rounded-xl` (12px)

#### 카드
- **기본**: `rounded-2xl shadow-md`
- **호버**: `hover:shadow-xl hover:-translate-y-1`
- **패딩**: `p-6`

#### 폼
- **입력 필드**: `border border-gray-300 rounded-lg px-4 py-2.5`
- **포커스**: `focus:ring-2 focus:ring-emerald-500 focus:ring-offset-2`
- **에러**: `border-red-500`
