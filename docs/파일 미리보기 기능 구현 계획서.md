#파일 미리보기 기능 구현 계획서

##1. 문서 개요

본 문서는 개인 및 공개 서비스용 **파일 미리보기 기능**을 구현하기 위한 기술적·법적·운영적 계획을 정리한 문서이다.

서비스 목적은 사용자가 업로드한 파일을 **브라우저에서 임시로 확인**할 수 있도록 돕는 것이며, 파일 저장·공유·배포 기능은 제공하지 않는다.

---

## 2. 구현 목표

- 프로그램 설치 없이 파일 내용을 빠르게 확인
- 개인용 및 공개 서비스 운영 가능 구조
- 저작권 및 법적 리스크 최소화
- 광고 및 향후 수익화 확장 가능

---

## 3. 지원 파일 형식 및 범위

| 파일 형식 | 미리보기 방식 | 비고 |
| --- | --- | --- |
| PDF | pdf.js 렌더링 | 우선 구현 |
| 이미지 (jpg/png/webp) | 브라우저 네이티브 | 기존 기능 연계 |
| Word (.docx) | HTML 변환 | 레이아웃 일부 제한 |
| Excel (.xlsx) | 테이블 렌더 | SheetJS 활용 |
| PowerPoint (.pptx) | 슬라이드 렌더 | 미리보기 수준 |
| HWP | PDF 변환 후 미리보기 | 서버 변환 시에만 제공 |

---

## 4. 기술 구현 confirms

### 4.1 기본 아키텍처 (권장)

```
사용자 브라우저
 ├─ 파일 선택
 ├─ 클라이언트 JS 파싱/렌더링
 └─ 서버로 파일 전송 ❌

```

- 서버는 HTML / JS 리소스만 제공
- 파일 데이터는 브라우저 메모리에서만 처리

---

### 4.2 사용 라이브러리

| 목적 | 라이브러리 | 라이선스 |
| --- | --- | --- |
| PDF 미리보기 | pdf.js | Apache 2.0 |
| Word 변환 | mammoth.js | BSD |
| Excel 파싱 | SheetJS (Community) | Apache 2.0 |
| PPT 렌더링 | pptxjs | MIT |
- 모든 라이브러리는 공개·상업 서비스 사용 가능
- 라이선스 고지는 About / License 페이지에 명시

---

## 5. 저작권 및 법적 안전 설계

### 5.1 기본 원칙

- 사용자가 업로드한 파일의 저작권은 사용자에게 있음
- 본 서비스는 콘텐츠를 저장·배포·공유하지 않음
- 파일은 미리보기 목적의 임시 처리만 수행

---

### 5.2 위험 요소 차단 설계

| 항목 | 정책 |
| --- | --- |
| 서버 저장 | 금지 |
| URL 접근 | 금지 |
| 공유 기능 | 제공하지 않음 |
| 파일 분석 | 광고/학습 목적 사용 금지 |
| 자동 삭제 | 브라우저 세션 종료 시 즉시 |

---

## 7. 사용자 고지 문구 (필수)

### 7.1 서비스 설명

```
본 서비스는 사용자가 업로드한 파일을
브라우저에서 임시로 미리보기 하기 위한 도구입니다.
파일은 서버에 저장되지 않습니다.

```

---

### 7.2 이용약관 핵심 문구

```
사용자가 업로드한 파일의 모든 권리는 사용자에게 있으며,
본 서비스는 파일을 저장하거나 제3자에게 제공하지 않습니다.

```

## 9. 결론

본 파일 미리보기 기능은 **클라이언트 기반 임시 처리 구조**를 유지할 경우,
저작권 및 법적 문제 없이 공개 서비스로 운영 가능하다.

광고 및 수익화 역시 **도구 제공 중심**으로 설계하면 안정적인 확장이 가능하다.
