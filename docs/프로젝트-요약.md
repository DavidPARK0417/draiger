# Draiger : 데일리 툴킷 – 프로젝트 요약

> 디지털 마케팅 전문가와 사업자를 위한 **데일리 비즈니스 툴 & 인사이트 플랫폼**

---

## 1. 서비스 한줄 정의

**Draiger : 데일리 툴킷**은 디지털 마케터와 사업자가 **마케팅 분석 · 실무 업무 · 인사이트 학습**을 한 곳에서 해결할 수 있도록 설계된 **웹 기반 종합 비즈니스 솔루션 플랫폼**입니다.  
Next.js 기반의 웹 앱 위에, 마케팅 계산 도구·업무 툴·콘텐츠·레시피를 통합 제공하여 **데이터 기반 의사결정과 일상 업무 효율화**를 돕습니다.

---

## 2. 주요 기능 개요

### 2.1 마케팅 분석 도구

디지털 마케팅의 핵심 지표를 빠르게 계산·비교할 수 있는 전문 분석 도구들을 제공합니다.

- **광고 성과 계산**: 여러 상품의 광고 성과를 비교하고 최적 상품 추천
- **키워드 분석**: 검색량·경쟁도·CPC 기반의 키워드 분석
- **ROI 계산기**: 투자 대비 수익률 계산 및 효율성 판단
- **손익분기점 계산기**: 고정비/변동비 기반 최소 판매 목표 계산
- **광고 예산 계산기**: 목표 전환수·CPC 기반 필요 예산 산출
- **CRO 전환율 최적화**: 전환율 개선 시 예상 매출 증가액 계산
- **수익성 진단**: CPA, LTV, LTV:CAC 비율을 3단계로 진단

> 모든 주요 분석 도구는 **Google Gemini AI**를 활용한 자동 추정·분석 기능과 연계됩니다.

### 2.2 실무/일상 업무 도구

마케터와 사업자가 자주 사용하는 반복 업무를 웹에서 바로 처리할 수 있도록 실용적인 도구를 제공합니다.

- **이미지 크기 조정**: 웹·광고용 이미지 리사이즈 및 최적화
- **파비콘 생성기**: 브랜드용 파비콘 자동 생성
- **QR 코드 생성기**: 링크·텍스트를 QR 코드로 변환
- **URL 단축기**: 긴 URL을 짧은 공유용 링크로 변환
- **글자수 세기**: 글자/단어 수 실시간 카운트
- **세계시간 변환기**: 글로벌 협업을 위한 시차 변환
- **알람 시계**: 업무 타이머·알람 설정
- **파일 미리보기**: PDF, Word, Excel, PPT, 이미지 등 다양한 파일 뷰어
- **이자 계산기**: 대출·예금 등 이자 계산

### 2.3 인사이트 & 레시피 콘텐츠

#### 인사이트(블로그) 콘텐츠

- **데이터 소스**: Notion 데이터베이스 기반 CMS
- **자동 갱신**: 인사이트 글을 매일 자동 동기화
- **주요 카테고리**  
  - 내일의 AI (AI 트렌드·활용 사례)  
  - 돈이 되는 소식 (비즈니스·투자 인사이트)  
  - 궁금한 세상 이야기 (사회 트렌드)  
  - 슬기로운 생활 (생활 꿀팁)  
  - 오늘보다 건강하게 (건강/웰빙)  
  - 마음 채우기 (심리·자기계발) 등
- **특징**: 마크다운 렌더링, TTS(음성 읽기), 카테고리/검색 기능

#### 오늘의메뉴 (레시피)

- Notion 레시피 데이터베이스를 사용하는 **요리 레시피 서비스**
- 레시피별 **재료 정보, 단계별 조리 가이드, 검색 기능** 제공
- Notion DB 변경만으로 자동으로 레시피가 사이트에 반영되도록 설계

## 3. 기술 및 아키텍처 개요

- **프레임워크**: Next.js 15 (App Router, 서버/클라이언트 컴포넌트 분리)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS 4 + 자체 디자인 시스템
- **주요 연동 서비스**
  - Google Gemini API: 마케팅 지표 추정·분석
  - Notion API: 인사이트/레시피 CMS
  - 네이버 검색광고 API: 키워드 검색량·CPC 조회
  - Nodemailer: 문의 메일 전송
- **기타**
  - Recharts: 분석 결과 시각화
  - Giscus: GitHub 기반 댓글
  - PWA (서비스워커, 웹 매니페스트, 설치 프롬프트)
  - 다크 모드 및 반응형 UI 지원

---

## 4. 플랫폼이 제공하는 가치

- **데이터 기반 의사결정**  
  감(Feeling)이 아니라 **지표와 시뮬레이션**을 기반으로 광고 전략·예산·상품 구성을 결정할 수 있습니다.

- **도구 + 콘텐츠의 결합**  
  단순 계산 도구가 아니라, **분석에 필요한 개념·사례·트렌드를 함께 제공**하여 “배우면서 쓰는” 경험을 제공합니다.

- **자동화와 효율화**  
  Notion·AI·API 연동을 통해 반복적인 작업을 자동화하고, 마케터·사업자가 **핵심 의사결정에만 집중**할 수 있도록 돕습니다.
