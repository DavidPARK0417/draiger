# 마케팅 도구 모음

다양한 마케팅 도구를 제공하는 웹 애플리케이션입니다. 광고 성과 분석, 예산 계산, 수익성 진단 등 마케팅에 필요한 다양한 계산 도구를 한 곳에서 제공합니다.

## 시작하기

### 1. 의존성 설치

```bash
pnpm install
```

### 2. 환경 변수 설정

프로젝트 루트에 `.env.local` 파일을 생성하고 다음 내용을 추가하세요:

```env
GEMINI_API_KEY=your_api_key_here
```

**Gemini API 키 발급 방법:**

1. [Google AI Studio](https://aistudio.google.com/app/apikey)에 접속
2. "Create API Key" 버튼 클릭
3. 생성된 API 키를 복사하여 `.env.local` 파일에 붙여넣기

### 3. 개발 서버 실행

```bash
pnpm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 제공되는 도구

### 1. 광고 성과 계산 도구

여러 상품의 광고 성과를 비교하고 최적의 상품을 찾습니다.

- **AI 기능**: 상품명만 입력하면 AI가 판매가, 순이익, 광고비, 전환수를 자동으로 추정합니다
- 여러 상품의 광고 성과를 한눈에 비교
- 최적의 상품 추천 및 분석

### 2. 키워드 분석 도구

키워드의 검색량, 경쟁도, CPC를 분석하여 최적의 키워드를 찾습니다.

- 키워드별 검색량 및 경쟁도 분석
- CPC(클릭당 비용) 추정
- AI 기반 키워드 추천

### 3. ROI 계산기

투자 대비 수익률을 계산하여 광고 효과를 측정합니다.

- ROI(투자 대비 수익률) 계산
- 광고 투자 대비 수익 분석
- AI 기반 ROI 추정 및 분석

### 4. 손익분기점 계산기

고정비와 변동비를 기반으로 손익분기점을 계산하여 최소 판매 목표를 설정합니다.

- 손익분기점 계산
- 최소 판매량 목표 설정
- AI 기반 손익분기점 분석

### 5. 광고 예산 계산기

목표 전환수와 CPC를 기반으로 필요한 광고 예산을 계산합니다.

- 목표 전환수 기반 예산 계산
- CPC 기반 예산 추정
- AI 기반 예산 최적화 제안

### 6. CRO 전환율 최적화 계산기

전환율 개선에 따른 월간/연간 예상 매출 증가액과 추가 확보 전환수를 계산합니다.

- 전환율 개선 효과 분석
- 예상 매출 증가액 계산
- AI 기반 전환율 최적화 제안

### 7. 마케팅 수익성 진단

목표 CPA, LTV, LTV:CAC 비율을 3단계로 진단하여 광고 예산을 최적화합니다.

- 3단계 수익성 진단
- CPA, LTV, LTV:CAC 비율 분석
- AI 기반 광고 예산 최적화 제안

## 추가 기능

### 문의하기

마케팅 도구의 오류나 문제점, 추가하고 싶은 기능이 있다면 언제든지 문의할 수 있습니다.

- 이메일을 통한 문의 전송
- 빠른 피드백 및 지원

## 기술 스택

- **Next.js 15** - React 프레임워크 (Turbopack 사용)
- **TypeScript** - 타입 안정성
- **Tailwind CSS 4** - 스타일링
- **Google Gemini AI** - 상품 정보 자동 추정 및 분석
- **Recharts** - 데이터 시각화 및 차트
- **Nodemailer** - 이메일 전송 기능

## 프로젝트 구조

```
src/
├── app/
│   ├── api/              # API 라우트
│   │   ├── analyze-*     # 분석 관련 API
│   │   ├── estimate-*    # 추정 관련 API
│   │   └── contact/      # 문의하기 API
│   ├── tools/            # 도구 페이지들
│   ├── contact/          # 문의하기 페이지
│   └── page.tsx          # 메인 페이지
└── components/
    └── Header.tsx        # 공통 헤더 컴포넌트
```

## 보안 고려사항

- `GEMINI_API_KEY`는 서버 사이드에서만 접근 가능합니다
- 클라이언트는 `/api/*` 엔드포인트를 통해 API를 호출합니다
- `.env.local` 파일은 Git에 커밋되지 않습니다 (`.gitignore`에 포함됨)
- 모든 API 키는 환경 변수로 관리되며 클라이언트에 노출되지 않습니다
